# Product List Management Application

A responsive frontend application built using **React** that allows users to manage a list of products with multiple views, search functionality, pagination, and in-memory data handling.

---

## üöÄ Features

### 1. Product List Display
- Displays products in:
  - **List View (Table)**
  - **Card View (Grid)**
- Toggle option to switch between List and Card views

---

### 2. Search Functionality
- Real-time search by **product name**
- Implements **500ms debounce** to improve performance

---

### 3. Add & Edit Product
- Form fields:
  - **Name** (required)
  - **Price** (number, required)
  - **Category** (required)
  - **Stock** (number)
  - **Description** (optional)
- Basic form validation with user-friendly error messages
- Product data is stored **in memory only** (no backend / API)

---

### 4. Pagination
- Pagination implemented for the product list
- Improves usability when handling multiple products

---

### 5. Responsive UI
- Mobile, tablet, and desktop friendly
- Clean and user-friendly interface
